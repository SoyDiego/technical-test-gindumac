# Technical Test - ReactJS - ExpressJS

## How to use...

1) Clone the project.

```
git clone https://github.com/SoyDiego/technical-test-gindumac
```
2) Install dependencies.

```
yarn install
```

3) Start project.

```
yarn start
```

4) Go to localhost.
```
http://localhost:5000/
```

## Technologies

-   ReactJS
-   Emotion (Style Components)
-   ExpressJS
